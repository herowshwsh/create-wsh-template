#  create-wsh-template


## 📖简介
- 用于快速搭建前端项目的命令行工具


## 📦 安装

```bash
npm install create-wsh-template -g
```
## 🚩使用

```bash
# 使用模板创建新项目
create-wsh-template create 

# 执行项目文件夹名称并选择模板创建新项目
create-wsh-template  create [projectName]

# 更新脚手架到最新版本
create-wsh-template  update

# 查看脚手架版本
create-wsh-template  -v
create-wsh-template  --version

```
